# NodeJs_Tchat
## Projet NodeJs

## 1) Téléchargement des modules

Lancer la commande :
```
npm install
```

## 2) Mettez en place la base de données

La solution la plus simple est d'importer le fichier "*tchat.sql*" dans PhpMyAdmin ou Adminer

## 3) Lancez le serveur web

Lancer la commande :
```
npm start
```

## 4) Accédez au tchat via son chemin local

Via http://localhost/Nodejs_tchat par éxemple.
**/!\** Attention, le serveur web démarré ne permet pas d'accéder au tchat, il hebèrge en revanche le serveur de sockets.
